# Schedule Manager

일정 관리를 위한 Qt 기반 데스크탑 프로그램입니다.  
업무 및 개인 일정을 통합 관리할 수 있도록 설계되었습니다.

---

## 주요 기능

- 일정 추가/수정/삭제
- 우선순위 및 카테고리별 분류
- 날짜, 제목, 카테고리 기반 상세 검색
- 파일 저장 및 불러오기 (JSON 포맷)
- 심플하고 직관적인 사용자 인터페이스

---

## 개발 환경

- **언어:** C++
- **프레임워크:** Qt 6
- **개발 툴:** Qt Creator

---

ScheduleManager/

 ├── ui/              # UI 설계 파일 (.ui)
 
 ├── resources/       # 이미지/아이콘 리소스
 
 ├── README.md        # 프로젝트 설명 파일
 
 └── (기타 빌드 설정 파일)

## 실행 방법

1. Qt Creator로 프로젝트 열기
2. 빌드(Build)
3. 실행(Run)

---

## 주의사항

- Qt 설치가 필요합니다 (Qt 6 버전 권장).
